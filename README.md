# Azure_dotNET-training
Azure上に、ASP.NETで作成したWebアプリを動作させる案件を担当することになったので、そのためのインプットを行います。  
主にASP.NETでのアプリ制作の実習をする予定です。  
ASPについては、デプロイの手順とか、各サービスの概要などをまとめていきます。
